# Example metadf

An example `metadf` table used to create an `EZbakRData` object. This
metadf describes the DCP2 KO dataset published in Luo et al., 2020.

## Usage

``` r
example_metadf
```

## Format

### `example_metadf`

A tibble with 6 rows and 3 columns

- sample:

  Sample name

- tl:

  Metabolic label feed time

- genotype:

  Whether sample was collected from WT or DCP2 KO cells

## References

Luo et al. (2020) Biochemistry. 59(42), 4121-4142
